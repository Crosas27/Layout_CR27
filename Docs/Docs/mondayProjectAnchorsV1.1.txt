--------------------------------------------------
CORE DESIGN PHILOSOPHY
--------------------------------------------------

Layout hierarchy:

RIB GRID → PANEL GRID → OPENINGS → CUTS

Rules:

- Rib grid is the source of truth.
- Panels are derived from rib spacing.
- Openings conform to panel layout.
- Cuts are derived from opening positions.

Panels dictate framing layout.
Openings DO NOT dictate panel layout.


--------------------------------------------------
SYSTEM GEOMETRY RULES
--------------------------------------------------

Panel Coverage Width:
36 inches (3 feet)

Rib Spacing:
12 inches

Panel Rib Pattern:

A — R — R — O

Where:

A = Seam Rib (anchor rib)
R = Mid Rib
O = Overlap Rib


Panel Rib Positions (from seam rib center):

Seam Rib:        0"
Mid Rib 1:       12"
Mid Rib 2:       24"
Overlap Rib:     36"


--------------------------------------------------
MEASUREMENT STANDARD
--------------------------------------------------

ALL FIELD MEASUREMENTS ORIGINATE FROM:

→ Rib Centers (Seam or Overlap)

Panel edges are NEVER used as reference points.

Rib centers define the true geometric grid.


--------------------------------------------------
OPENING SYSTEM RULES (v1)
--------------------------------------------------

Openings are defined by:

- Start position (from nearest seam/overlap rib center)
- Width

Opening measurements:

Rib Center → Jamb Edge

System behavior:

- Engine does NOT auto-shift openings
- Engine does NOT re-layout panels for openings
- Openings are user-authoritative inputs

Minimum clearance (soft rule):

6 inches from nearest rib

(Warning system may be implemented later)


--------------------------------------------------
LAYOUT START RULES
--------------------------------------------------

Primary anchor point:

Outside Corner

Layout direction:

- Left → Right  
OR  
- Right → Left

Layout NEVER begins from wall center.


--------------------------------------------------
FIELD MARKING METHOD
--------------------------------------------------

Layout marks placed at:

→ Every seam rib (36" intervals)

Marks represent:

- Panel start positions
- Layout control references

Marks DO NOT represent:

- Screw lines
- Rib positions


--------------------------------------------------
SYMMETRY RULE
--------------------------------------------------

When wall length is not divisible by 36":

leftover = wall_length % 36

End panel width:

(36 + leftover) / 2

Rules:

- Start panel = End panel
- Layout must remain visually balanced
- Symmetry overrides full-panel preference


--------------------------------------------------
OFFSET RULE (FUTURE SUPPORT)
--------------------------------------------------

Offset is applied to:

→ Rib Grid ONLY

Purpose:

- Adjust layout to avoid conflicts
- Maintain rib alignment control

Offset does NOT:

- Break rib spacing
- Override symmetry rules
